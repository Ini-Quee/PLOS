# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in PLOS, please report it
responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email: ericainiking.3000@gmail.com.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

I will respond within 48 hours and work with you to resolve the issue
before any public disclosure.
